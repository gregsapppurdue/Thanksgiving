## Form Questions

The RSVP form includes the following questions:

### Required Questions (*)

1. **Can you attend?** *
   - Yes, I'll be there
   - Sorry, can't make it

2. **Is this your 1st Thanksgiving Meal?** *
   - Yes
   - No

3. **What are the names of people attending?** *
   - Text input field

4. **Please provide phone number for WhatsApp group** *
   - Text input field
   - Note: Updates and details will be shared there

### Optional Questions

5. **Please list any side items, drinks, or desserts that you will be bringing**
   - Text input field

6. **Please list any dietary restrictions or allergies**
   - Text input field

7. **Provide any additional comments or notes to the host**
   - Text input field

## Notes

- All questions marked with * are required fields
- The form is embedded in the web application at the RSVP section
- Responses are automatically collected in Google Sheets
- The WhatsApp phone number is used for group communication and event updates
